Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

chodzi o to, żeby usunąć wszystkie białe spacje z podanego adresu text, sprawdzić, czy wszystkie znaki razem nie są cyframi i czy każdy znak jest pisany wielkimi literami.

Ok, od czego powinienem zacząć?

Przede wszystkim proszę zastąpić wszystkie białe spacje w text pustą linią "". Proszę to zrobić metodą str.replace(). Proszę przypisać wynik do zmiennej. 

text1 = text.replace(" ", "")
To proste. Co powinienem sprawdzić w następnej kolejności?

Musi Pan sprawdzić kilka warunków. W tym celu można użyć operatora and lub funkcji all() z wszystkimi niezbędnymi warunkami jako elementami sekwencji argumentów. 

return all((...))

Proszę zauważyć, że funkcja przyjmuje tylko jeden argument - sekwencję warunków.

Zacznijmy sprawdzać nasz nowy tekst1! Co powinno być pierwsze?

Zgodnie z opisem, sprawdzenie pustego text1 musi zwrócić False. Tak więc, aby uzyskać True, trzeba text1 NIE być pusty. Jest to pierwsze sprawdzenie i aby je wykonać, wystarczy umieścić sam text1 jako argument all(). Python interpretuje niepuste ciągi jako True. 

return all((text1, ...))
Krótko i sprytnie! Co dalej?

Teraz proszę sprawdzić, czy text1 nie jest liczbą całkowitą (nie ma liter i może być przekonwertowana na liczbę całkowitą). Proszę użyć metody str.isdigit(), która sprawdza, czy wszystkie znaki w ciągu znaków są cyframi. Proszę odwrócić wynik sprawdzania, ponieważ NIE potrzebują Państwo, aby text1 było liczbą. 

return all((text1, not text1.isdigit(), ...))
Jeśli tekst1 nie jest pusty i nie jest liczbą, jak sprawdzić, czy jest pisany wielkimi literami?

Tak, to jest ostateczne sprawdzenie. Python ma specjalną metodę do sprawdzania tego (cyfra z dużą literą jest również uważana za dużą, tak samo z małą literą) str.isupper(). Zwraca ona True jeśli wszystkie symbole w ciągu są pisane wielkimi literami. 

return all((text1, not text1.isdigit(), text1.isupper()))
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Proszę tylko opakować swój kod w wyrażenie funkcyjne. Zaleca się używanie podpowiedzi typu dla danych wejściowych i wyjściowych.

Oto całe rozwiązanie:

def is_all_upper(text: str) -> bool:

    text1 = text.replace(" ", "")
    return all((text1, not text1.isdigit(), text1.isupper()))

Teraz, gdy rozumieją Państwo wewnętrzną logikę sprawdzania, chcę Państwu powiedzieć, że Python jest wystarczająco inteligentny i str.isupper() sam wykonuje wszystkie niezbędne sprawdzenia! Tak więc, poniższy kod działa poprawnie. 

def is_all_upper(text: str) -> bool:

    return text.isupper()

Mogą Państwo również użyć funkcji lambda (anonimowej): 

is_all_upper = lambda text: text.isupper()

Ponieważ używa Pan standardowej metody bez żadnych zmian, może Pan użyć jej zamiast swojej funkcji w ten sposób. Wszystkie te rozwiązania działają. 

is_all_upper = str.isupper

Proszę zauważyć, że nie ma nawiasów - odnosi się Pan do metody, a nie ją wywołuje.
