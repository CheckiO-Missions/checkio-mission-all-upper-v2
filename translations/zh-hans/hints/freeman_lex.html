我不知道如何开始解决这项任务。

💡 总体思路是从给定的text 中删除所有空格，检查所有字符是否为数字，每个字符是否为大写字母。

好吧，我应该从什么开始呢？

首先，用空行""替换text 中的所有空格。用str.replace() 方法完成。将结果赋值给一个变量。 

text1 = text.replace(" ", "")
这很简单。下一步我应该检查什么？

您需要检查几个条件。为此，您可以使用and 运算符或all() 函数，将所有必要条件作为参数序列的元素。 

return all((...))

注意，函数只需要一个参数--条件序列。

让我们开始检查我们的新文本1！第一个应该是什么？

根据描述，检查空的text1 必须返回False 。因此，要得到True ，您需要text1 不是空的。这是第一次检查，执行它只需要将text1 本身作为all() 的参数。Python 将非空字符串解释为True 。 

return all((text1, ...))
简短而聪明！下一步是什么？

现在您需要检查text1 是否不是实数（没有字母，可以转换为整数）。使用str.isdigit() 方法，检查字符串中的所有字符是否为数字。您需要反转检查结果，因为您不需要您的text1 是数字。 

return all((text1, not text1.isdigit(), ...))
如果text1不为空且不是数字，如何检查它是否为大写字母？

是的，这是最后的检查。Python 有一个特定的方法来检查 (大写字母的数字也被认为是大写字母，小写字母也一样)str.isupper() 。如果字符串中的所有符号都是大写，它返回True 。 

return all((text1, not text1.isdigit(), text1.isupper()))
那么，我的最终解决方案应该是怎样的呢？

您只需将代码封装为函数表达式。强烈建议对输入和输出数据使用类型提示。

以下是整个解决方案：

def is_all_upper(text: str) -> bool:

    text1 = text.replace(" ", "")
    return all((text1, not text1.isdigit(), text1.isupper()))

现在，当你理解了检查的内在逻辑，我想告诉你，Python 是足够聪明的，str.isupper() 自己完成所有必要的检查！因此，下面的代码可以正常工作。 

def is_all_upper(text: str) -> bool:

    return text.isupper()

您也可以使用 lambda (匿名) 函数： 

is_all_upper = lambda text: text.isupper()

由于您使用的是标准方法，没有做任何改动，您可以使用它来代替您的函数。所有这些解决方案都有效。 

is_all_upper = str.isupper

请注意，这里没有括号--您指的是方法，而不是调用它。
