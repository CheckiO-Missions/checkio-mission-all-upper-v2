Ok, what should I start from?

First of all, replace all white spaces in text with empty line "". Do it with str.replace() method. Assign the result to a variable. 

text1 = text.replace(" ", "")
That's easy. What should I check next?

You need to check a few conditions. For this purpose you may use and operator or all() function with all necessary conditions as elements of argument sequence. 

return all((...))

Notice, that the function takes only one argument - a sequence of conditions.

Let's start checking our new text1! what should be the first one?

According to the description, the checking of empty text1 must return False. So to get True, you need text1 NOT to be empty. This is the first checking and to perform it you just need to put text1 itself as an argument of all(). Python interprets non-empty strings as True. 

return all((text1, ...))
Short and clever! What's next?

Now you need to check if text1 is not a solid number (has no letters and may be converted to an integer). Use str.isdigit() method, which checks all characters in a string to be digits. You need to invert the result of checking since you do NOT need your text1 to be a number. 

return all((text1, not text1.isdigit(), ...))
If text1 is not empty and is not a number, how to check if it's in upper case?

Yes, this is the final checking. Python has a specific method to check it (digit with a letter in upper case is considered as upper too, the same with letter in lower case) str.isupper(). It returns True is all symbols in a string are in upper case. 

return all((text1, not text1.isdigit(), text1.isupper()))
So, how my final solution should look like?

You just need to wrap your code into function expression. Using type hints for input and output data are highly recommended.

Here is the whole solution:

def is_all_upper(text: str) -> bool:

    text1 = text.replace(" ", "")
    return all((text1, not text1.isdigit(), text1.isupper()))

Now, when you understand the inner logic of checking, I want to tell you, that Python is smart enough and str.isupper() does all necessary checking by itself! So, the following code works just fine. 

def is_all_upper(text: str) -> bool:

    return text.isupper()

You may also use lambda (anonymous) function: 

is_all_upper = lambda text: text.isupper()

Since you use a standard method without any changes, you may use it instead of your function this way. All these solutions work. 

is_all_upper = str.isupper

Notice, that there are no parenthesis - you refer to the method, not calling it.
